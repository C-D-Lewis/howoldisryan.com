# howoldisryan.com

He may be older than you think.

## Play the scroller greeting

```shell
cd scroller

npm ci

node index.js
```

## Build the site

```shell
cd howoldisryan.com

npm ci

npm run build
```

Then serve it locally:

```shell
python3 -m http.server
```

Then view it at `http://localhost:8000`.
